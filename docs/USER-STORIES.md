# User Stories | داستان‌های کاربری

**سامانه ارزیابی و نظارت و مدیریت اماکن** — فاز ۲  
**قالب:** `As a [نقش], I want to [فعالیت], so that [فایده]`

---

## دانشجو

| # | داستان کاربری | اولویت |
|---|---------------|--------|
| US-S01 | As a **دانشجو**, I want to **جستجو و فیلتر اماکن ورزشی دانشگاهم را ببینم**, so that **سریع‌تر مکان مناسب تمرین یا رویداد را پیدا کنم**. | P0 |
| US-S02 | As a **دانشجو**, I want to **درخواست رزرو با تاریخ و ساعت مشخص ثبت کنم**, so that **زمان استفاده از مکان را به‌صورت رسمی رزرو کنم**. | P0 |
| US-S03 | As a **دانشجو**, I want to **قبل از ثبت، هشدار تداخل زمانی یا ظرفیت را ببینم**, so that **درخواست ردنشده ارسال نکنم**. | P0 |
| US-S04 | As a **دانشجو**, I want to **رزروهای آینده و گذشته خود را در یک صفحه ببینم**, so that **برنامه‌ام را مدیریت کنم**. | P0 |
| US-S05 | As a **دانشجو**, I want to **رزرو را حداقل ۲۴ ساعت قبل از شروع لغو کنم**, so that **در صورت تغییر برنامه، ظرفیت مکان آزاد شود**. | P0 |
| US-S06 | As a **دانشجو**, I want to **پس از پایان رزرو، کیفیت مکان را امتیازدهی کنم**, so that **به بهبود خدمات ورزشی دانشگاه کمک کنم**. | P0 |

---

## مدیر ورزش دانشگاه

| # | داستان کاربری | اولویت |
|---|---------------|--------|
| US-U01 | As a **مدیر ورزش دانشگاه**, I want to **درخواست‌های رزرو در انتظار را در تقویم و فهرست ببینم**, so that **به‌موقع تأیید یا رد کنم**. | P0 |
| US-U02 | As a **مدیر ورزش دانشگاه**, I want to **رزرو را با ذکر دلیل رد کنم**, so that **دانشجو دلیل تصمیم را بداند**. | P0 |
| US-U03 | As a **مدیر ورزش دانشگاه**, I want to **اطلاعات و وضعیت اماکن دانشگاه را ثبت و ویرایش کنم**, so that **فهرست ملی اماکن به‌روز بماند**. | P0 |
| US-U04 | As a **مدیر ورزش دانشگاه**, I want to **میانگین رضایت هر مکان را ببینم**, so that **اولویت نگهداری و تعمیر را تعیین کنم**. | P1 |

## دبیر منطقه‌ای

| # | داستان کاربری | اولویت |
|---|---------------|--------|

## مدیر ملی

| # | داستان کاربری | اولویت |
|---|---------------|--------|
| US-N01 | As a **مدیر ملی**, I want to **شاخص‌های کلان (تعداد اماکن، بهره‌برداری، رضایت) و گزارش مقایسه مناطق را ببینم**, so that **سیاست‌گذاری ورزش دانشگاهی را داده‌محور انجام دهم**. | P0 |
